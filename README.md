## 📄 是什么

WebCopilot 是一个基于 Vue3 + TS + Vite + Manifest V3 的开源浏览器插件

-   划词AI辅助（翻译、解释、搜索...）
-   即刻搜索
-   宗旨是提高您的浏览器体验

## 📦 安装依赖

```bash
pnpm install
```

## 🚀🔨 开发和构建

```bash
pnpm build
```

1. 在浏览器地址中输入：edge://extensions/ 或 chrome://extensions/
2. 打开开发人员模式
3. 点击加载解压缩的扩展导入构建后的build文件即可
